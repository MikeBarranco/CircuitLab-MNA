Bienvenido a CircuitLab MNA

Tu herramienta educativa para el Análisis Nodal Modificado.

Este proyecto te permite resolver circuitos eléctricos lineales usando el **Análisis Nodal Modificado (MNA)**, un método poderoso y sistemático ideal para la solución por computadora.

¿Qué es el Análisis Nodal Modificado (MNA)?

El MNA es un método formal que genera un sistema de ecuaciones para cualquier circuito. [cite_start]A diferencia del análisis nodal simple, el MNA maneja fácilmente fuentes de voltaje, lo que lo hace más robusto para los programas de simulación. [cite: 14, 15]

Todo el análisis se reduce a resolver una ecuación matricial fundamental:

Ax = z

Matriz A (El Circuito)

Es la matriz de coeficientes. [cite_start]Contiene toda la información sobre cómo están conectados los componentes (resistencias, capacitores, etc.) y las fuentes de voltaje. [cite: 178, 569]

Vector x (Las Incógnitas)

Es el vector de incógnitas. [cite_start]Contiene los voltajes de cada nodo y las corrientes que fluyen a través de las fuentes de voltaje que queremos encontrar. [cite: 185, 499]

Vector z (Las Fuentes)

Es el vector de "fuentes conocidas". [cite_start]Contiene los valores de las fuentes de corriente y las fuentes de voltaje independientes que alimentan el circuito. [cite: 189, 503]

Características del Simulador

  • Análisis DC y AC: Resuelve circuitos en corriente directa (f=0) y corriente alterna (f>0).
  • Componentes Soportados:
    • Resistores (R)
    • Fuentes de Voltaje (V)
    • Fuentes de Corriente (I)
    • Capacitores (C)
    • Inductores (L)
  • Resultados Completos: Muestra voltajes nodales, corrientes de fuentes y las matrices (A, x, z) que tu profesora revisará.
  • Lógica Robusta: El código maneja casos complejos, como circuitos sin fuentes de voltaje, gracias a su lógica condicional.
