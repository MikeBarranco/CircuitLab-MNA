¿Qué es el Análisis Nodal Modificado (MNA)?

El MNA es un método formal que genera un sistema de ecuaciones para cualquier circuito. A diferencia del análisis nodal simple, el MNA maneja fácilmente fuentes de voltaje, lo que lo hace más robusto para los programas de simulación.

Todo el análisis se reduce a construir y resolver una única ecuación matricial:

Ax = z

A (La Matriz del Circuito)

Contiene toda la información sobre CÓMO están conectados los componentes (conductancias) y las fuentes de voltaje.

x (El Vector de Incógnitas)

Contiene los valores que queremos encontrar: los voltajes de cada nodo y las corrientes de las fuentes de voltaje.

z (El Vector de Fuentes)

Contiene los valores conocidos que "alimentan" el circuito: los valores de las fuentes de corriente y de voltaje.

¿Cómo se Construyen las Matrices? (El Algoritmo)

La magia del MNA es que sigue reglas sistemáticas (un algoritmo) para llenar las matrices A y z. La gran matriz A se construye a partir de 4 sub-matrices más pequeñas:

Imagen conceptual de la construcción de la matriz A.
  • Matriz G (Conductancias): Se llena con las resistencias e impedancias.
    • Diagonales (G[k,k]): Suma de todas las conductancias (1/R) conectadas al nodo 'k'.
    • Off-diagonales (G[k,j]): Negativo de la conductancia ( -1/R ) entre el nodo 'k' y el nodo 'j'.
  • Matriz B (Incidencia de Fuentes V): Se llena con `+1` y `-1`.
    • Si el terminal positivo de `V1` se conecta al nodo `k`, `B[k,1]` es `+1`.
    • Si el terminal negativo de `V1` se conecta al nodo `j`, `B[j,1]` es `-1`.
  • Matriz C (Transpuesta): Es simplemente la matriz B "volteada" (transpuesta).
  • Matriz D (Ceros): Es una matriz de ceros (ya que solo usamos fuentes independientes).

Nuestro simulador construye estas cuatro matrices "detrás de cámaras", las ensambla en la matriz A y resuelve el sistema `Ax=z` para encontrar los voltajes y corrientes.
