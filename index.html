¿Qué es el Análisis Nodal Modificado (MNA)?

El MNA es un método formal que genera un sistema de ecuaciones para cualquier circuito. A diferencia del análisis nodal simple, el MNA maneja fácilmente fuentes de voltaje, lo que lo hace más robusto para los programas de simulación.

Todo el análisis se reduce a resolver una ecuación matricial fundamental:

Ax = z

A (La Matriz del Circuito)

Contiene toda la información sobre CÓMO están conectados los componentes (conductancias) y las fuentes de voltaje.

x (El Vector de Incógnitas)

Contiene los valores que queremos encontrar: los voltajes de cada nodo y las corrientes que fluyen a través de las fuentes de voltaje.

z (El Vector de Fuentes)

Contiene los valores conocidos que "alimentan" el circuito: los valores de las fuentes de corriente y las fuentes de voltaje.

¿Cómo se Construyen las Matrices?

La magia del MNA es que sigue reglas sistemáticas. La gran matriz A se construye a partir de 4 sub-matrices más pequeñas: G, B, C y D.

[Espacio reservado para la imagen de la construcción de la matriz A, como en los PDFs]

  • Matriz G (Conductancias): Se llena con los valores de las resistencias (y la impedancia de capacitores/inductores). Es la parte principal del análisis nodal.
  • Matriz B (Incidencia): Se llena con `+1` y `-1` para marcar dónde están conectadas las fuentes de voltaje.
  • Matriz C (Transpuesta): Es simplemente la matriz B "volteada" (transpuesta).
  • Matriz D (Ceros): Es una matriz de ceros (ya que solo usamos fuentes independientes).

Nuestro simulador construye estas cuatro matrices "detrás de cámaras" y las ensambla en la matriz A para resolver el circuito.
