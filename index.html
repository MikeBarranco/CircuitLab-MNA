¿Qué es el Análisis Nodal Modificado (MNA)?

El Modified Nodal Analysis es una técnica sistemática para analizar circuitos eléctricos que resulta en un sistema de ecuaciones lineales algebraicas. A diferencia del análisis nodal clásico, el MNA puede manejar directamente fuentes de voltaje sin necesidad de crear supernodos.

El método es especialmente valioso porque es algorítmico y predecible, lo que lo hace ideal para la implementación computacional en simuladores de circuitos profesionales como SPICE.

Ecuación Fundamental del MNA

Todo circuito analizado con MNA se reduce a resolver el sistema:

A × x = z
A

Matriz de coeficientes (n+m)×(n+m) que contiene información sobre la topología y elementos del circuito

x

Vector de incógnitas (n+m)×1 con voltajes nodales y corrientes en fuentes de voltaje

z

Vector de términos conocidos (n+m)×1 con valores de fuentes de corriente y voltaje

Donde n = número de nodos (excluyendo el nodo de referencia o tierra), y m = número de fuentes de voltaje independientes.

Estructura de las Matrices del MNA

La matriz A y el vector z tienen una estructura específica que se construye siguiendo reglas algorítmicas precisas:

Matriz A - Estructura por Bloques

G
B
C
D

Matriz G (Conductancias)

Dimensión: n × n

Contenido: Conductancias de elementos pasivos (R, C, L)

  • Diagonal: G[i][i] = suma de conductancias conectadas al nodo i
  • Off-diagonal: G[i][j] = -g si hay conductancia g entre nodos i y j

Fórmulas de conductancia:

• Resistor: g = 1/R

• Capacitor (AC): g = jωC

• Inductor (AC): g = 1/(jωL)

Matriz B (Incidencia de Fuentes V)

Dimensión: n × m

Contenido: Conexión de fuentes de voltaje a nodos

  • B[i][j] = +1 si el terminal positivo de la fuente j está en el nodo i
  • B[i][j] = -1 si el terminal negativo de la fuente j está en el nodo i
  • B[i][j] = 0 en cualquier otro caso

Matriz C (Transpuesta)

Dimensión: m × n

Contenido: C = BT (transpuesta de B)

Esta matriz asegura la conservación de corriente (Ley de Kirchhoff) en las fuentes de voltaje.

Matriz D (Fuentes Dependientes)

Dimensión: m × m

Contenido: Matriz de ceros para fuentes independientes

En circuitos con solo fuentes independientes, D es una matriz de ceros m×m.

Vectores x y z

Vector x (Incógnitas)

v₁ v₂ ⋮ vₙ
i_V₁ i_V₂ ⋮ i_Vₘ

Primera parte (n elementos): Voltajes de nodos v₁, v₂, ..., vₙ medidos respecto al nodo de referencia (tierra)

Segunda parte (m elementos): Corrientes que fluyen a través de las fuentes de voltaje i_V₁, i_V₂, ..., i_Vₘ

Vector z (Términos Conocidos)

i₁ i₂ ⋮ iₙ
e₁ e₂ ⋮ eₘ

Primera parte (n elementos): Corrientes inyectadas por fuentes de corriente en cada nodo

Segunda parte (m elementos): Valores de voltaje de las fuentes de voltaje e₁, e₂, ..., eₘ

Algoritmo MNA Paso a Paso

El algoritmo del Análisis Nodal Modificado se ejecuta en 5 pasos sistemáticos. Cada paso construye una parte específica del sistema de ecuaciones:

1

Identificar Nodos y Elementos

  • Numerar todos los nodos del circuito (0, 1, 2, ..., N-1)
  • Designar un nodo como nodo de referencia (tierra, V = 0)
  • Contar: n = nodos totales - 1 (sin contar tierra)
  • Contar: m = número de fuentes de voltaje independientes
  • Listar todos los elementos con sus nodos de conexión y valores
2

Construir Matriz G (Conductancias)

Inicializar: Matriz G de tamaño n×n con todos los elementos en 0

Para cada elemento pasivo (R, C, L) entre nodos i y j:

  1. Calcular su conductancia g:
    • Resistor: g = 1/R
    • Capacitor (AC): g = jωC donde ω = 2πf
    • Inductor (AC): g = 1/(jωL)
  2. Sumar g a G[i][i] (si i ≠ tierra)
  3. Sumar g a G[j][j] (si j ≠ tierra)
  4. Restar g de G[i][j] y G[j][i] (si ambos ≠ tierra)
3

Construir Matrices B, C y D

Matriz B (n×m):

  • Para cada fuente de voltaje j conectada entre nodos i+ y i-:
  • B[i+][j] = +1 (si i+ ≠ tierra)
  • B[i-][j] = -1 (si i- ≠ tierra)

Matriz C (m×n): C = BT (transpuesta de B)

Matriz D (m×m): D = matriz de ceros (para fuentes independientes)

4

Construir Vector z

Vector i (n×1) - Corrientes de fuentes de corriente:

  • Para cada nodo k: i[k] = suma de corrientes inyectadas
  • Positivo si la corriente entra por el terminal + de la fuente
  • Negativo si la corriente sale por ese nodo

Vector e (m×1) - Voltajes de fuentes de voltaje:

  • e[j] = valor de la fuente de voltaje j

Concatenar: z = [i; e] (vector de tamaño (n+m)×1)

5

Resolver el Sistema A·x = z

Ensamblar matriz A:

A = [G B]
    [C D]

Resolver: Usar eliminación gaussiana, factorización LU, o método matricial

Resultado: El vector x contiene:

  • Primeros n elementos: voltajes de nodos v₁, v₂, ..., vₙ
  • Últimos m elementos: corrientes en fuentes de voltaje i_V₁, ..., i_Vₘ

Ejemplos Prácticos Resueltos

A continuación presentamos tres ejemplos completos que ilustran diferentes aspectos del MNA: circuitos con fuentes de corriente, circuitos con fuentes de voltaje, y análisis AC con elementos reactivos.

Ejemplo 1: Circuito con Fuentes de Corriente

Descripción del Circuito

Elementos del circuito:

  • Fuente de corriente Is = 2 A (del nodo 0 al nodo 1)
  • Resistor R1 = 2 Ω (del nodo 1 al nodo 0)
  • Resistor R2 = 4 Ω (del nodo 1 al nodo 2)
  • Resistor R3 = 1 Ω (del nodo 2 al nodo 0)
  • Fuente de corriente Is2 = 4 A (del nodo 0 al nodo 2)

Nodo de referencia: Nodo 0 (tierra)

Incógnitas: v1 y v2 (voltajes de los nodos 1 y 2)

Solución Paso a Paso

Paso 1: Identificación

n = 2 nodos (sin contar tierra: nodos 1 y 2)

m = 0 fuentes de voltaje

Sistema resultante: 2 ecuaciones con 2 incógnitas

Paso 2: Matriz G (2×2)

Conductancias:

g1 = 1/R1 = 1/2 = 0.5 S

g2 = 1/R2 = 1/4 = 0.25 S

g3 = 1/R3 = 1/1 = 1.0 S

Construcción de G:

G[1][1] = g1 + g2 = 0.5 + 0.25 = 0.75

G[2][2] = g2 + g3 = 0.25 + 1.0 = 1.25

G[1][2] = G[2][1] = -g2 = -0.25

G = [0.75 -0.25]
    [-0.25 1.25]
Paso 3: No hay fuentes de voltaje

Como m = 0, no hay matrices B, C ni D

El sistema se simplifica a: G·v = i

Paso 4: Vector i (2×1)

i[1] = corriente inyectada en nodo 1 = 2 A

i[2] = corriente inyectada en nodo 2 = 4 A

i = [2]
    [4]
Paso 5: Resolver el Sistema

Sistema de ecuaciones:

0.75·v1 - 0.25·v2 = 2
-0.25·v1 + 1.25·v2 = 4

Solución:

v1 = 4.0 V

v2 = 4.0 V

Ejemplo 2: Circuito con Fuente de Voltaje

Descripción del Circuito

Elementos del circuito:

  • Fuente de voltaje Vs = 10 V (del nodo 1 al nodo 0)
  • Resistor R1 = 2 Ω (del nodo 1 al nodo 2)
  • Resistor R2 = 3 Ω (del nodo 2 al nodo 0)

Nodo de referencia: Nodo 0 (tierra)

Incógnitas: v1, v2, y iVs

Solución Paso a Paso

Paso 1: Identificación

n = 2 nodos (nodos 1 y 2)

m = 1 fuente de voltaje

Sistema resultante: 3 ecuaciones con 3 incógnitas (2 voltajes + 1 corriente)

Paso 2: Matriz G (2×2)

Conductancias:

g1 = 1/R1 = 1/2 = 0.5 S

g2 = 1/R2 = 1/3 ≈ 0.333 S

Construcción de G:

G[1][1] = g1 = 0.5

G[2][2] = g1 + g2 = 0.5 + 0.333 = 0.833

G[1][2] = G[2][1] = -g1 = -0.5

G = [0.5 -0.5 ]
    [-0.5 0.833]
Paso 3: Matrices B, C, D

Matriz B (2×1):

La fuente Vs tiene su terminal + en nodo 1 y terminal - en nodo 0

B[1][1] = +1 (nodo positivo)

B[2][1] = 0 (no conectada)

B = [1]
    [0]

Matriz C (1×2): C = BT

C = [1 0]

Matriz D (1×1):

D = [0]
Paso 4: Vector z (3×1)

Vector i: No hay fuentes de corriente

i[1] = 0

i[2] = 0

Vector e:

e[1] = 10 V

z = [0 ]
    [0 ]
    [10]
Paso 5: Ensamblar y Resolver A·x = z

Matriz A (3×3):

A = [0.5 -0.5 1]
    [-0.5 0.833 0]
    [1 0 0]

Solución del sistema:

v1 = 10.0 V

v2 = 6.0 V

iVs = -2.0 A (la fuente suministra 2 A)

Verificación:

Corriente en R1: iR1 = (v1 - v2)/R1 = (10 - 6)/2 = 2 A

Corriente en R2: iR2 = v2/R2 = 6/3 = 2 A

Ejemplo 3: Circuito AC con Capacitor

Descripción del Circuito

Elementos del circuito:

  • Fuente de voltaje AC Vs = 120 V @ 60 Hz (del nodo 1 al nodo 0)
  • Resistor R = 100 Ω (del nodo 1 al nodo 0)
  • Capacitor C = 26.5 µF (del nodo 1 al nodo 0)

Frecuencia: f = 60 Hz

Nodo de referencia: Nodo 0 (tierra)

Incógnitas: v1 (complejo) y iVs (complejo)

Solución Paso a Paso

Paso 1: Identificación

n = 1 nodo (nodo 1)

m = 1 fuente de voltaje

ω = 2πf = 2π(60) = 376.99 rad/s

Paso 2: Matriz G (1×1) con impedancias complejas

Conductancia del resistor:

gR = 1/R = 1/100 = 0.01 S

Conductancia del capacitor (compleja):

gC = jωC = j(376.99)(26.5×10-6)

gC = j(0.00999) ≈ j0.01 S

Matriz G total:

G[1][1] = gR + gC = 0.01 + j0.01

G = [0.01 + j0.01]
Paso 3: Matrices B, C, D

B = [1], C = [1], D = [0]

Paso 4: Vector z
z = [0 ]
    [120]
Paso 5: Resolver el Sistema

Matriz A (2×2):

A = [0.01+j0.01 1]
    [1 0]

Solución:

v1 = 120 + j0 V (por definición de la fuente)

iVs = -(0.01 + j0.01)(120) = -1.2 - j1.2 A

En forma polar:

|iVs| = √(1.2² + 1.2²) = 1.697 A

∠iVs = arctan(1.2/1.2) = -135° (la fuente suministra corriente con desfase)

Interpretación física:

El capacitor introduce un desfase de 90° adelantado, y junto con el resistor resulta en una impedancia total de magnitud |Z| = 120/1.697 = 70.7 Ω con ángulo de -45°.
